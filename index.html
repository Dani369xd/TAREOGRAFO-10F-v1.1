TE DAMOS LA BIENVENIDA A LA PAGINA OFICIAL DEL TAREOGRAFO DE 10F

esta pagina se creo con el fin de que todo estudiante del curso de 10F tenga acceso a una informacion diaria de lo que se hace en cada clase, para de esta manera tener un soporte extra de ayuda como ventaja a la hora de presentar trabajos, realizar tareas, exposisiones, quizes, talleres, evaluaciones ETC. (todo esto de manera legal, sin problemas o concecuencias de ser visto como "TRAMPA!" por parte de criticos)

Acerca de esta pagina (creditos, proceso de creacion, ETC)

INFORMACION, creditos

Desde un inicio, esta pagina fue gracias a HTML, con soporte de GOOGLE, DRIVE, CLASSROOM ETC...la cual fue la encargada de crear y desarroyar todo el software de esta aplicacion y pagina que fue creada solo para los estudiantes del grado de 10F, esta pagina estara en mantenimiento cada dia (devido a que es actualizable) y por ende se sube contenido cada dia en el cual cada estudiante, padre etc pueda ver cuantas veces quiera para ayudarse en el proceso academico de el año 2024. Esta app y pagina tambien cuenta con una IA, hecha con el fin de realizar actualizaciones cada 24 horas, haciendo que de esta manera el creador no tenga que mover "ni un solo dedo" para realizar este ardua y dificil tarea que requiere de mucha dedicacion, aunque de vez en cuando tiene que entrar para realizar alguna novedad o arreglar un error que puedan reportar los usuarios, y nuevamente recordamos que ESTA PAGINA ESTA DISPONIBLE HASTA EL ULTIMO DIA DEL AÑO, demostrando que esta pagina fue una propuesta FIEL y bien PROMETIDA por parte del representante.
